# README Banner Template

Copy this to the top of any non-canonical repo's README.md:

```markdown
> **Status:** [Active / Archived / Mirror / Experimental]  
> **Canonical Repository:** [MQGT-SCF](https://github.com/Cbaird26/MQGT-SCF)  
> **Purpose:** [One sentence describing what this repo is for]  
> **Not for:** [One sentence about what this repo is NOT for - prevents misinterpretation]

---

[Rest of README continues below]
```

## Examples

### Mirror/Archive
```markdown
> **Status:** Archived  
> **Canonical Repository:** [MQGT-SCF](https://github.com/Cbaird26/MQGT-SCF)  
> **Purpose:** Historical snapshot of early ToE development.  
> **Not for:** Active development or current research - see canonical repo.

---
```

### App/Experimental
```markdown
> **Status:** Experimental  
> **Canonical Repository:** [MQGT-SCF](https://github.com/Cbaird26/MQGT-SCF)  
> **Purpose:** Application layer for MQGT-SCF theory (trading, simulations, etc.).  
> **Not for:** Core theory development - see canonical repo.

---
```

### Active Library
```markdown
> **Status:** Active  
> **Canonical Repository:** [MQGT-SCF](https://github.com/Cbaird26/MQGT-SCF)  
> **Purpose:** Python library for MQGT-SCF computations and analysis.  
> **Not for:** Theory papers or experimental data - see canonical repo.

---
```
