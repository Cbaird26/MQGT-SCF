# Neutrino Channel Documentation

## Overview

Tests scalar fields via neutrino oscillation probability modifications.

## Methodology

- **Mass Matrix**: Maps to neutrino mass matrix modifications
- **Oscillation Prob**: Maps to probability modifications
- **Data Sources**: Super-K, SNO, KamLAND, Daya Bay

## Implementation

- Module: `code/inference/neutrinos/oscillation_modifications.py`
- Script: `scripts/generate_neutrino_bounds.py`
- Data: `data/constraints/neutrinos/`

## Sensitivity

- m_c ~ 10⁻³ - 10⁻¹ eV (neutrino mass scale)
