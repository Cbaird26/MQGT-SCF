# Dark Matter Channel Documentation

## Overview

Tests scalar fields via dark matter direct detection (if Φ_c is DM candidate).

## Methodology

- **Scattering Rate**: Maps to DM-nucleus scattering rates
- **Data Sources**: XENON, LUX, PandaX, LZ

## Implementation

- Module: `code/inference/dark_matter/direct_detection.py`
- Script: `scripts/generate_dm_bounds.py`
- Data: `data/constraints/dark_matter/`

## Sensitivity

- m_c ~ 1 GeV - 1 TeV (if dark matter)

## Note

Only relevant if Φ_c is a dark matter candidate.
