## Why You Might Not See PDFs

**GitHub doesn't render PDFs inline.** When you click a PDF file on GitHub, you'll see:
- A preview page (if the file is small)
- Or just metadata

**To actually view/download:**
1. Click the "Download" button (top right of the file view)
2. Or use the direct "raw" links above
3. Or clone the repository: `git clone https://github.com/Cbaird26/MQGT-SCF.git`
